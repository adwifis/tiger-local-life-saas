# 产品说明

## 当前产品

```text
虎鲸本地生活商家营销助手
```

## 产品目标

在 1 周内交付一个可商业化上线的 AI SaaS 首发版，服务：

- 本地生活门店老板
- 店长
- 代运营团队
- 城市代理

## 核心价值

- 5 分钟生成可直接发的营销内容
- 模板贴合餐饮、美业、健身、教培等本地生活场景
- 代理可批量服务多个客户门店
- 平台可按套餐和额度快速售卖

## 首发范围

- 商家端：门店资料、模板中心、AI 生成、历史记录、额度
- 代理端：客户管理、代运营、额度查看
- 平台端：模板、套餐、代理、商家管理

## 当前仓库状态

已具备：

- Next.js + Prisma + Docker 开发底座
- 百炼环境变量与校验脚本
- 安全检查和部署脚本
- PRD、开发计划、运维文档

待补齐：

- 正式业务数据模型
- 模板库落库
- 百炼生成服务
- 代理工作台
- 套餐与额度业务逻辑
